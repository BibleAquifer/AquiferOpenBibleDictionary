## Lidah Api (id: 369135)

LIDAH API
=========

Sebuah frasa dalam [Kisah 2:3](https://ref.ly/Acts2:3), menggambarkan penampilan fisik Roh Kudus. Lidah\-lidah api tampaknya memenuhi nubuatan Yohanes Pembaptis bahwa Dia Yang Akan Datang akan membaptis dengan Roh Kudus dan api ([Mat. 3:11](https://ref.ly/Matt3:11); [Luk. 3:16](https://ref.ly/Luke3:16)). Para murid disebutkan penuh dengan Roh Kudus, menggenapi janji Perjanjian Lama yang diulang oleh Yohanes Pembaptis dan Yesus tentang baptisan Roh.

*Lihat juga* Pentakosta.

* **Associated Passages:** MAT 3:11; LUK 3:16; ACT 2:3
